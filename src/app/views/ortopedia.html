Orthopedics Store Webpage

I was hired to develop and deployed the official website for an orthopedic store. The website features a product catalog, contact form, and a blog section. It was deployed on a shared hosting server and is maintained by the client

Project Purpose and Goal

Designed so that customers can review the catalog of products that the store offers and can carry out transactions.

Web Stack and Explanation

For this project, I used the Grav CMS, which is a flat-file CMS that allows you to create websites without the need for a database. I used the Quark theme as a base and customized it to the client's needs. I also used the Form plugin to create the contact form and the Blog plugin to create the blog section. The UX/UI design was done by me, and I implemented it using HTML, CSS, and JavaScript. Finally, I deployed the site to Heroku
