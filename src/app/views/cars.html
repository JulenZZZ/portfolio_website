Web Stack and Explanation

I used Angular to create a single-page application that allows users to view a list of cars and their details. The application uses Angular's built-in routing and HTTP client modules to fetch data from a JSON file and display it on the page. I also used Angular Material to create a responsive and visually appealing user interface. The application is deployed to GitHub Pages for easy access and sharing.
